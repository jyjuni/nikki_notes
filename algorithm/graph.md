# 图

## 原理

### 无向图和有向图

入度：指向当前点的边

出度：从当前点出发的边

无向图每个点的的入度=出度

### 邻接表/邻接矩阵

两种结构各有优劣

- 邻接表可以直接查出一个节点的所有边

- 邻接矩阵可以查出有多少边

技巧：

自己实现一个熟悉的结构模板，遇到不同的表达图的数据结构只需要写数据方式之间的转换，然后调自己的结构

结构：graph、node、edge
